# Cursor Dark Theme

This file contains theme settings for the cursor-dark mode.

## Colors
A collection of colors defined by their hex code.
- Background color: #1B1F23
- Foreground color: #CCCCCC
- Primary color: #4F8EF7
- Secondary color: #9DA5B4
- Border color: #313842
- Surface color: #252B32
- Accent color: #4F8EF7
- Muted color: #6E7681

## Spacing
Defines spacing levels used throughout the theme.
- xs: 4.0 units
- sm: 8.0 units
- md: 16.0 units
- lg: 24.0 units
- xl: 32.0 units
- xxl: 48.0 units

## Radius
Defines border radius levels for elements.
- sm: 4.0 units
- md: 6.0 units
- lg: 8.0 units
- full: 9999.0 (full width)

## Typography
Defined font styles and sizes used throughout the theme.
- Body: Font Family Inter, Size 13px, Line Height 18px
- Heading: Font Family Inter, Size 18px, Line Height 24px, Weight 500
- Caption: Font Family Inter, Size 11px, Line Height 14px, Weight 400
- Mono: Font Family JetBrainsMono, Size 13px, Line Height 18px, Weight 400

## Status Icons
Defines colors for status icons.
- Success: #3FB950 (Green)
- Warning: #D29922 (Yellow)
- Error: #F85149 (Red)
- Info: #58A6FF (Blue)
